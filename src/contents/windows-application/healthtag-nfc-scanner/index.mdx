โปรแกรมที่ใช้เชื่อมต่อกับ NFC Reader ของ Silicon craft ADR12 ที่ HealhTAG ใช้งาน โดยเครื่องอ่านจะเชื่อมต่อผ่านทาง COM port

## ฟีเจอร์

- สามารถใช้งานได้บนระบบปฏิบัติการย้อนหลังไปถึง Windows 7 SP1 (แต่ต้องติดตั้ง Update ตาม [How to Update Windows 7 All at Once with Microsoft’s Convenience Rollup](https://www.howtogeek.com/255435/how-to-update-windows-7-all-at-once-with-microsofts-convenience-rollup/))
- ส่วนติดต่อผู้ใช้งานมีทั้งภาษาไทยและภาษาอังกฤษ
- สามารเลือกบราว์เซอร์ในการเปิด Link ที่อ่านจาก NFC Tag ได้

## ขั้นตอนการใช้งาน

1. เสียบสาย USB Type-C กับเครื่องอ่านและเสียบสาย USB Type-A กับคอมพิวเตอร์ จะได้ยินเสียง Beep และไม่จำเป็นต้องติดตั้ง Driver ใดๆ เพิ่มเติม
2. เปิดโปรแกรมการนั้นเลือก NFC Reader ที่ต้องการเชื่อมต่อ

## เบื้องหลังการพัฒนา

เป็นการปรับปรุงโปรแกรมเดิมให้มีความสวยงามและใช้งานง่ายมากขึ้น

โดยพัฒนาโดยใช้ Microsoft Visual Studio 2022 Preview โดยใช้ภาษา C# และใช้ UI Framework คือ WPF (Windows Presentation Foundation) เนื่องจากต้องการให้มันสามารถใช้งานได้ใน Windows 7 SP1 โดยใช้ Lib ชื่อ [WPF-UI ](https://github.com/lepoco/wpfui) และ [ReactiveUI](https://www.reactiveui.net/)
